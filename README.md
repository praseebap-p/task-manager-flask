# Task Manager Web App

A simple web application built using Flask to manage daily tasks.

## Features
- Add tasks
- Delete tasks
- Simple user interface

## Tech Stack
- Python (Flask)
- HTML

## Project Structure
task-manager-flask/
│
├── app.py
├── templates/
│   └── index.html

## How to Run
1. Install Flask:
   pip install flask  
2. Run the app:
   python app.py  
3. Open browser:
   http://127.0.0.1:5000/
